# Hostel Complaint Management System

A web-based Hostel Complaint Management System developed using Flask and SQLite.

## Features

* Student Complaint Registration
* Anonymous Complaint Reporting
* Admin Login Authentication
* Complaint Status Tracking
* Search Complaints by Room Number
* Dashboard Statistics
* Complaint Resolution Workflow
* Complaint Deletion Functionality

## Tech Stack

* Python
* Flask
* SQLite
* HTML
* CSS

## Project Workflow

1. Student registers a complaint.
2. Complaint is stored in the database.
3. Admin logs into the dashboard.
4. Admin updates complaint status.
5. Complaint progresses from:

   * Pending
   * In Progress
   * Resolved
